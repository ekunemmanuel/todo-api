Welcome to the API

Available endpoints:

v1

The v1 API version offers basic CRUD operations for managing tasks in memory.

  • GET /api/v1/tasks - Retrieve a list of tasks

  • POST /api/v1/tasks - Create a new task

    {
    "title": "string",
    "completed": "boolean"
}

  • PUT /api/v1/tasks/[id] - Update an existing task by ID

    {
    "title": "string",
    "completed": "boolean"
}

  • DELETE /api/v1/tasks/[id] - Delete a task by ID

v2

The v2 API version introduces Firestore integration for persistent storage of tasks.

  • GET /api/v2/tasks - Retrieve a list of tasks

  • POST /api/v2/tasks - Create a new task

    {
    "title": "string",
    "completed": "boolean"
}

  • PUT /api/v2/tasks/[id] - Update an existing task by ID

    {
    "title": "string",
    "completed": "boolean"
}

  • DELETE /api/v2/tasks/[id] - Delete a task by ID

  • POST /api/v2/tasks/[id] - Create a task with a specific ID

    {
    "title": "string",
    "completed": "boolean"
}

v3

The v3 API version extends the capabilities of v2 with authentication and query support. It includes login and registration endpoints, and supports query parameters for filtering and ordering tasks. A global rate limiter has been added for enhanced security.

  • POST /api/v3/auth/login - User login

    {
    "email": "string",
    "password": "string"
}

  • POST /api/v3/auth/register - User registration

    {
    "email": "string",
    "password": "string",
    "firstName": "string",
    "lastName": "string"
}

  • GET /api/v3/tasks - Retrieve a list of tasks with filtering and ordering options (returns ID in descending order by default based on createdAt)

    Query Parameters:

    completed (boolean) - Filter tasks by completion status

  • POST /api/v3/tasks - Create a new task

    {
    "title": "string",
    "completed": "boolean"
}

  • PUT /api/v3/tasks/[id] - Update an existing task by ID

    {
    "title": "string",
    "completed": "boolean"
}

  • DELETE /api/v3/tasks/[id] - Delete a task by ID

  • POST /api/v3/tasks/[id] - Create a task with a specific ID

    {
    "title": "string",
    "completed": "boolean"
}
